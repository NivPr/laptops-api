Laptops Routes

Method URL Body Description
GET /laptops Get a list of all the laptops, (10 results per page)
GET /laptops/?s=Apple search by req.query.s value
GET /laptops/?page=2 Show page by req.query.page value, (10 results per page)
GET /laptops/os/linux Show laptops by os name, (10 results per page)
GET /laptops/prices/?min=2000&max=5000 Get a list of all the laptops priced between min value and max value
